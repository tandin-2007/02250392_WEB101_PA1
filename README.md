## 📌 Project Overview
This project is a React-based web application where I recreated a webpage from an existing application using a **component-based architecture**.

The goal of this assignment is to demonstrate understanding of:
- React fundamentals
- Component structure
- Responsive design
- Modern frontend development practices

As required in the assignment instructions :contentReference[oaicite:0]{index=0}, the project focuses on breaking down a webpage into reusable components and building a responsive UI.

---

## 🚀 Technologies Used
- React.js / Next.js
- Tailwind CSS (for styling)
- JavaScript (ES6+)
- HTML5 & CSS3

---

## 🧱 Component Structure
The application is divided into reusable components:

- Navbar → Navigation bar  
- Sidebar → Menu/navigation section  
- Feed → Main content area  
- Card / Post Component → Reusable UI element  
- Footer → Bottom section  

Each component follows the **Single Responsibility Principle**.
